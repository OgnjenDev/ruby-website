About Ruby OS

My name is Ognjen Đokić and I am 11 years old. Since I was little, I’ve been fascinated by computers, operating systems, and programming. I loved exploring how things work “behind the screen,” and that’s how I came up with the idea to create my own operating system. Ruby OS is my personal project – a lightweight, simple, and educational operating system that can even run on very old computers. I built it using Python, aiming to make it useful and fun, and to inspire other young people to believe they can also create something great, regardless of their age. Ruby OS includes basic apps like a terminal, Notepad, and Paint, it has a start menu and user space, and I constantly improve it and add new features. This project is my way of showing that with effort, knowledge, and imagination, big ideas can come to life — even at the age of 11.

Features

  • Terminal with commands
  • Start menu
  • Notepad and Paint applications
  • File manager and user space
  • Boot process
  • Log system, API support (coming soon)
  • Ability to extend with new applications

System Requirements

Minimum Requirements:

  • Processor: 500 MHz (x86 or ARM)
  • RAM: 128 MB
  • Disk: 200 MB free space
  • Resolution: 800x600
  • Input: keyboard and mouse (or touchscreen)
  • Python 3.7+

Recommended Specifications:

  • Processor: 1 GHz+ (2 cores)
  • RAM: 512 MB or more
  • Disk: 1 GB free space
  • Resolution: 1024x768 or higher
  • Internet connection for updates
  • Audio support (coming soon)

Note: For the best experience, we recommend using Linux systems

Installation Steps

  1. Download the ZIP file.
  2. Extract it to your desired location.
  3. Run EBOOT.py with Python 3.7+.
  4. Enjoy Ruby OS!
